很多人第一次听说 eDiary，是从父辈的旧电脑里翻出来的安装包，或者是某篇"被遗忘的良心软件"合集文章。这款诞生于 1999 年的**eDiary 电子日记本**，到现在已经走过二十多个年头，依然有人在用、有人在更新、有人在它上面写了几百万字。

简单说，eDiary 是一款专注本地记录与隐私加密的日记软件，支持 Windows、macOS、Android 三端，核心卖点是云同步、Markdown 编辑和深度加密，开发方是国内的白日梦工作室。这句话基本就是它的全部定义，不用再绕弯子。

## 这玩意儿到底解决了什么问题

写日记这件事，说起来简单，做起来麻烦的点其实在"安全感"上。

发朋友圈，怕被熟人看见。存在手机备忘录，怕换手机丢了。存网盘文档，又总觉得不踏实——万一被看到怎么办？

eDiary 的逻辑反而朴素：本地优先存储，密码即加密种子，连开发者自己都解不开你的数据。云同步只是锦上添花的功能，不是必须依赖的东西。这种"离线优先、云端可选"的架构，跟现在动不动就要联网才能用的软件比，确实是个异类。

不过话说回来，这种"老派"也是双刃剑——它不会帮你做情绪分析，不会给你画心情曲线，更不会用算法猜你想写什么。它就是个本本分分的记录工具。

## 实际用起来是什么体验

打开软件的第一感觉是：轻。安装包不到 10MB，启动几乎不卡顿，跟那些动辄几百兆、内存常驻几个 G 的"生产力工具"完全不是一个量级。

编辑体验接近 Word，字体、排版、表格、图片插入都齐全，现在还加上了 Markdown 和双链功能，写作的时候可以在普通文本和 Markdown 之间随意切换。

时间轴管理是另一个亮点。所有日记按时间线排列，回顾某年某月发生的事情，滑动就能找到，不用翻文件夹。

> "eDiary 一直保持着极少资源占用、单个 exe 文件即可运行，这是非常难得的。"——一位使用多年的老用户在官网留言区这样评价。

至于云同步，目前**仅支持 Windows、macOS 和安卓设备间的数据互通，暂不支持 iOS 端**，这点如果你是苹果手机用户，提前要心里有数。

## 免费版够用吗？什么时候该升级会员

这是大部分人最纠结的问题。

老实讲，**免费版的本地记录功能是完整的**。写日记、加密、模板、时间轴管理，这些核心功能不需要花一分钱。真正限制免费用户的，是云同步、多端数据互通和 AI 写作助手这三项。

2. 是不是经常需要换设备、重装系统？
3. 想不想用 AI 帮你续写、总结或润色文字？

三个问题里有两个答案是"是"，那升级会员基本就划算了。否则免费版足够用很久。

## 值不值得买？看这三类人

eDiary 不是大而全的生产力工具，定位反而很窄——专注记录这一件事。所以它特别适合：

- 习惯每天写点东西、需要长期坚持的人
- 在意隐私、不想把私人记录放在云笔记或社交平台的人
- 不需要复杂数据库功能，只想要一个干净的时间轴工具的人

如果你需要的是团队协作、知识图谱或者复杂的多媒体笔记管理，eDiary 可能不是最优解，Notion 或 Obsidian 这类工具会更合适。但单纯论"写日记"这件事，它做得足够专注。

## 常见问题 FAQ

**eDiary 支持 iOS 吗？**
目前不支持。eDiary 可在 Windows、macOS、Android 三端运行并互相同步数据，iOS 版本暂未上线。

**eDiary 是免费软件还是收费软件？**
本地记录功能完全免费,云同步、AI 写作助手等高级功能需要开通标准会员或永久会员才能使用。

**永久会员和标准会员最大的区别是什么？**
除了使用时长上"年限"与"永久"的差异，永久会员还多了私有云部署服务，云存储空间则是 1GB（标准会员为 3GB）。

**eDiary 的数据安全吗？密码忘了能找回吗？**
本地与云端都采用基于用户密码的加密机制,官方本身也无法解密数据，因此一旦忘记密码，无法通过官方找回，建议提前把密码记录在安全的地方。

**正版渠道购买和官网直接购买有什么区别？**
授权代理渠道通常因批量采购成本更低,价格会比官网零售价更实惠,同时享有本地售后与技术咨询支持，比起来源不明的低价激活码也更有保障。
